# Naive Bayes README

## Running the NoteBook
### Virtual environment
Change to Naive Bayes directory:  
`cd 'Naive Bayes'`

#### On MacOS/Linux:  
Create venv:  
`python -m venv .venv`  

Start venv:  
`source .venv/bin/activate`   

#### On Windows:  
Create venv:  


Start venv:
`.venv/Scripts/activate`

#### Install Required Packages
`pip install -r requirements.txt`
